# allure_backend_theme

- If you use Project module then use our 'allure_backend_theme_project' module to resolve project pivot view.

Installation steps for 'allure_backend_theme_project':
1. Get a zip file of 'allure_backend_theme_project' module from 'allure_backend_theme' module source.
2. Extract it in your addons.
3. Restart the server.
4. Go to menu 'Apps' and click on 'Update Apps List' > 'Update'.
5. Search for 'allure_backend_theme_project' module and install it.
